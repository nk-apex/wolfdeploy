Create a single-page web application that functions like Heroku for deploying WhatsApp bots. The app should:

1. **UI Components Required:**
   - Use the provided shadcn/ui Command component for bot selection
   - Clean, modern dashboard layout
   - Dark mode support
   - Responsive design

2. **Core Functionality:**
   - Users can select from available WhatsApp bots
   - Each bot has an app.json configuration file defining required env vars (session ID and phone number)
   - Users input their WhatsApp session ID and phone number
   - Click "Deploy" to start deployment on a VPS
   - Real-time deployment logs
   - Status tracking (deploying, running, stopped, failed)
   - Auto-generated deployment URL

3. **Technical Implementation:**
   - Frontend: React with Tailwind CSS
   - Backend API endpoints:
     - GET /api/bots - List available bots
     - GET /api/bots/:id/app.json - Fetch bot configuration
     - POST /api/deploy - Start new deployment
     - GET /api/deploy/:id/logs - Stream deployment logs
     - GET /api/deployments - List user deployments
     - POST /api/deploy/:id/stop - Stop deployment
     - DELETE /api/deploy/:id - Delete deployment

4. **Deployment Flow:**
   - User searches/selects bot from command palette
   - Bot details page shows requirements from app.json
   - User fills in required env vars (session ID, phone number)
   - System validates inputs
   - Deployment starts with progress indicator
   - Real-time logs show Docker build and deployment process
   - Success page shows deployment URL and management options

5. **VPS Integration:**
   - Use Docker for containerization
   - Each deployment gets unique port/subdomain
   - Nginx reverse proxy for routing
   - Automatic SSL via Let's Encrypt
   - Resource limits per deployment

6. **Session Management:**
   - Secure storage of WhatsApp session data
   - Session persistence across restarts
   - QR code display for initial authentication
   - Session backup and restore

7. **Additional Features:**
   - One-click deploy from GitHub repo
   - Environment variable management
   - Deployment metrics (CPU, memory, uptime)
   - Automatic restarts on crash
   - Logs viewer with search
   - Domain management
